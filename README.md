# VinuChain Ecosystem

A modern, responsive showcase for projects building on VinuChain. Features tabs for Audited/Upcoming/Launched/Live Utilities, search, smooth animations, and black & white design. Built with Next.js 15, TypeScript, Tailwind CSS.

## Features
- **Tabbed Navigation**: Filter by project stage.
- **Search & Filter**: Real-time project search.
- **Responsive Design**: Mobile-first, dark theme.
- **Smooth Animations**: CSS keyframes for card entry/hover.
